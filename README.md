# Dark Mode Memory Game

A simple browser-based memory matching game with a dark mode theme toggle.

## Features

- 4x4 and 6x6 grid modes
- Timer and move counter
- Star rating based on number of moves
- Dark mode toggle with saved preference
- Local high score persistence using `localStorage`

## How to Run

1. Open `memory-game.html` in your browser.
2. Choose a grid size.
3. Click or press Enter/Space on cards to flip them.
4. Match all pairs to win and see your score.

## Notes

- The game stores best scores locally in your browser.
- Dark mode preference is saved between sessions.

## Files

- `memory-game.html` — the full game implementation in a single HTML file.
